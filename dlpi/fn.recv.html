Expand description

Receive a message from a DLPI link.

Data is placed into provided buffer. Return values is (address bytes read, message bytes read).

If no message is received within msec milliseconds, returns ResultCode::ETimedout.

src must be at least sys::DLPI_PHYSADDR_MAX in length.
